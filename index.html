📋 Sheet 1 — Computer Basics: True / False
S1·Q1 True/False The calculating device called the Pascaline could calculate sums up to eight figures long.
✅ ANSWER
True — The Pascaline (invented by Blaise Pascal, 1642) could calculate sums up to eight figures.
S1·Q2 True/False All programs must be loaded into the CPU before they can be executed and all data must be loaded into main memory before it can be manipulated.
✅ ANSWER
False — Programs must be loaded into main memory (RAM), not the CPU, before execution.
S1·Q3 True/False Main memory is an ordered sequence of cells and each cell has a random location in main memory.
✅ ANSWER
False — Main memory is an ordered sequence of cells and each cell has a unique (not random) address/location.
S1·Q4 True/False The program that loads first when you turn on your computer is called the operating system.
✅ ANSWER
True — When the computer powers on, the BIOS/firmware loads the operating system first.
S1·Q5 True/False Analog signals represent information with a sequence of 0s and 1s.
✅ ANSWER
False — Digital signals represent information with 0s and 1s; analog signals are continuous.
S1·Q6 True/False The machine language is a sequence of 0s and 1s.
✅ ANSWER
True — Machine language (binary code) consists entirely of 0s and 1s that the CPU directly understands.
S1·Q7 True/False A binary code is a sequence of 0s and 1s.
✅ ANSWER
True — A binary code is by definition a sequence of binary digits (0s and 1s).
S1·Q8 True/False A sequence of eight bits is called a byte.
✅ ANSWER
True — By definition, 8 bits = 1 byte.
S1·Q9 True/False One GB is 2^20 MB.
✅ ANSWER
False — One GB is 2^10 MB (1024 MB), not 2^20 MB.
S1·Q10 True/False In ASCII, A is the 65th character.
✅ ANSWER
True — In ASCII, the character 'A' has the decimal value 65.
S1·Q11 True/False The number system used by a computer is base 2.
✅ ANSWER
True — Computers use the binary (base-2) number system internally.
S1·Q12 True/False An assembler translates the assembly language instructions into machine language.
✅ ANSWER
True — An assembler converts assembly language mnemonics into machine code (binary).
S1·Q13 True/False A compiler translates the source program into an object program.
✅ ANSWER
True — A compiler translates high-level source code into an object (machine-code) program.
S1·Q14 True/False In a C++ program, statements that begin with the symbol # are called preprocessor directives.
✅ ANSWER
True — Lines beginning with # (like #include, #define) are preprocessor directives processed before compilation.
S1·Q15 True/False An object program is the machine language version of a high-level language program.
✅ ANSWER
True — The object program is the machine-code output produced by the compiler from the source code.
S1·Q16 True/False All logical errors, such as division by 0, are reported by the compiler.
✅ ANSWER
False — Logical errors (like dividing by zero at runtime) are NOT caught by the compiler; they are runtime or logic errors.
S1·Q17 True/False In object-oriented design (OOD), a program is a collection of interacting objects.
✅ ANSWER
True — In OOD, a program is designed as a set of objects that interact with each other.
S1·Q18 True/False An object consists of data and operations on that data.
✅ ANSWER
True — In OOP, an object encapsulates data (attributes) and the operations (methods) that act on that data.
S1·Q19 True/False ISO stands for International Organization for Standardization.
✅ ANSWER
True — ISO stands for International Organization for Standardization, which standardizes C++.

The Preprocessor processes all statements beginning with # before the compiler translates the code.
S1·Q27 Short Answer What is programming?
✅ ANSWER
Programming is the process of giving a set of instructions (a recipe) to a computer to solve a problem or perform a specific task.
S1·Q28 Short Answer What is an algorithm?
✅ ANSWER

✅ ANSWER
The problem-solving process involves: 1) Analyze the problem, 2) Design an algorithm, 3) Implement (code) the algorithm in a programming language, 4) Test and debug the program, 5) Maintain the program.

S2·Q1 Algorithm Design an algorithm to find the perimeter and area of a rectangle.
✅ ANSWER — Algorithm
Input: length, width
Steps:
1. Get the length of the rectangle.
2. Get the width of the rectangle.
3. Calculate perimeter = 2 × (length + width)
4. Calculate area = length × width
5. Output perimeter and area.

S2·Q2 Algorithm Design an algorithm that calculates the sales tax and price of an item. The state's portion (stateSalesTax) is 4%, city's (citySalesTax) is 1.5%. If the item is a luxury item (e.g., a car > $50,000), there is a 10% luxury tax (luxuryTax).
✅ ANSWER — Algorithm
1. Get the original price of the item.
2. Calculate stateSalesTax = price × 0.04
3. Calculate citySalesTax = price × 0.015
4. If price > 50000 then luxuryTax = price × 0.10, else luxuryTax = 0
5. totalTax = stateSalesTax + citySalesTax + luxuryTax
6. finalPrice = price + totalTax
7. Output totalTax and finalPrice.
S2·Q3 Algorithm Design an algorithm for monthly paycheck of a salesperson. Every salesperson has a base salary. Bonus: ≤5 years = $10/year; >5 years = $20/year. Additional commission: sales ≥$5000 and <$10000 = 3%; sales ≥$10000 = 6%.
✅ ANSWER — Algorithm
1. Get baseSalary, yearsWorked, totalSales.
2. If yearsWorked ≤ 5 then bonus = yearsWorked × 10
   Else bonus = yearsWorked × 20
3. If totalSales ≥ 10000 then commission = totalSales × 0.06
   Else if totalSales ≥ 5000 then commission = totalSales × 0.03
   Else commission = 0
4. paycheck = baseSalary + bonus + commission
5. Output paycheck.

✅ ANSWER — Algorithm

2. Calculate circumference = 2 × π × radius  (π ≈ 3.14159)

S2·Q5 Algorithm There are 10 students in a class. Each student has taken five tests (each worth 100 pts). Design an algorithm to calculate the grade for each student and the class average. Grading: ≥90=A, ≥80=B, ≥70=C, ≥60=D, else F.
✅ ANSWER — Algorithm
1. Initialize classTotal = 0
2. Repeat for each of the 10 students:
  a. Get student's name and 5 test scores.
  b. sum = test1+test2+test3+test4+test5
  c. average = sum / 5
  d. Assign grade: if avg≥90 → A; elif avg≥80 → B; elif avg≥70 → C; elif avg≥60 → D; else F
  e. Output student name, average, grade.
  f. classTotal += average
3. classAverage = classTotal / 10
4. Output classAverage.
S2·Q6 Algorithm Design an algorithm to convert the change given in quarters, dimes, nickels, and pennies into pennies. (Quarter=25¢, Dime=10¢, Nickel=5¢, Penny=1¢)
✅ ANSWER — Algorithm
1. Get the number of quarters.
2. Get the number of dimes.
3. Get the number of nickels.
4. Get the number of pennies.
5. totalPennies = (quarters×25) + (dimes×10) + (nickels×5) + (pennies×1)

S2·Q7 Algorithm Design an algorithm to find the real roots of a quadratic equation of the form ax² + bx + c = 0, where a, b, and c are real numbers and a is nonzero.
✅ ANSWER — Algorithm
1. Get a, b, c.
2. Calculate discriminant = b² − 4ac
3. If discriminant > 0: two real roots → x1 = (−b + √disc) / (2a), x2 = (−b − √disc) / (2a)
4. If discriminant = 0: one real root → x = −b / (2a)
5. If discriminant < 0: no real roots (complex roots)
6. Output the result.
S2·Q8 Algorithm + C++ Statements The volume of a sphere is (4.0/3.0)πr³ and the surface area is 4.0πr². Given the radius, design an algorithm that computes the volume and surface area of the sphere. Also write the C++ statements (assume π = 3.141592).
✅ ANSWER
Algorithm:
1. Get the radius.
2. Calculate volume = (4.0/3.0) × 3.141592 × radius³
3. Calculate surfaceArea = 4.0 × 3.141592 × radius²
4. Output volume and surfaceArea.

C++ Code: 
double radius, volume, surfaceArea;
const double PI = 3.141592;
cin >> radius;
volume = (4.0/3.0) * PI * radius * radius * radius;
surfaceArea = 4.0 * PI * radius * radius;
cout << "Volume = " << volume << endl;
cout << "Surface Area = " << surfaceArea << endl;
S2·Q9 Algorithm Given the radius (in inches) and price of a pizza, design an algorithm to find the price of the pizza per square inch.

S6·Q4 Programming Write a C++ code to find the real roots of a quadratic equation ax² + bx + c = 0, where a, b, c are real numbers and a is nonzero.
✅ ANSWER — C++ Code
#include <iostream>
#include <cmath>
using namespace std;
int main() {
    double a, b, c, disc, x1, x2;
    cout << "Enter a, b, c: "; cin >> a >> b >> c;
    disc = b*b - 4*a*c;
    if (disc > 0) {
        x1 = (-b + sqrt(disc)) / (2*a);
        x2 = (-b - sqrt(disc)) / (2*a);
        cout << "Two real roots: x1=" << x1 << ", x2=" << x2 << endl;
    } else if (disc == 0) {
        x1 = -b / (2*a);
        cout << "One real root: x=" << x1 << endl;
    } else {
        cout << "No real roots (discriminant is negative)." << endl;
    }
    return 0;
}
S6·Q5 Programming Write a C++ code to compute the volume and surface area of a sphere. Volume = (4.0/3.0)πr³, Surface Area = 4.0πr². The code must check that radius is not negative; if it is, display "Error!".
✅ ANSWER — C++ Code
#include <iostream>
#include <cmath>
using namespace std;
int main() {
    const double PI = 3.14159265;
    double radius, volume, surfaceArea;
    cout << "Enter radius: "; cin >> radius;
    if (radius < 0) {
        cout << "Error!" << endl;
    } else {
        volume      = (4.0/3.0) * PI * pow(radius, 3);
        surfaceArea = 4.0 * PI * pow(radius, 2);
        cout << "Volume       = " << volume      << endl;
        cout << "Surface Area = " << surfaceArea << endl;
    }
    return 0;
}

📌 Section 1 — Programming Basics, Output, Comments
Sec1·Q1 Concept What is programming? Provide a simple definition.
✅ ANSWER
Programming is simply giving a recipe (a set of instructions) to the computer to perform a specific task. It is the process of giving instructions to a computer to solve a problem or perform a task. Note: The computer is extremely fast, but it is "dumb" — it will only do exactly what you tell it to do.
Sec1·Q2 Concept How does the computer understand us? What is the difference between high-level language and machine code?
✅ ANSWER
We speak natural languages (Arabic/English), but computers only understand binary (0s and 1s). We need a "translator".
• High-Level Language: Human-readable (like C++)
• Machine Code: Computer-readable (010101)
• Compiler: The translator that converts C++ code into machine code.
Sec1·Q3 Concept What is an IDE? Why do we use it?
✅ ANSWER
IDE (Integrated Development Environment) is a special text editor that serves as a "workspace" where we write, compile (translate), and run our code. Examples: Visual Studio, Code::Blocks, Dev-C++.

Sec1·Q7 Concept What is the difference between endl and \n?
✅ ANSWER
Both \n and endl move the cursor to the next line. However:
• \n is an escape sequence character; it is faster and most commonly used.
• endl also flushes the output buffer (forces immediate output display), which is slightly slower.
Recommendation: prefer \n for performance.
Sec1·Q8 Concept What are the 6 steps in processing a C++ program? Name each step and its output file.
✅ ANSWER
Step 1 — Editor: Source Code (.cpp) — raw code written by programmer.
Step 2 — Preprocessor: Expanded Source Code (.i) — processes #include etc.
Step 3 — Compiler: Object File (.obj/.o) — checks syntax, translates to machine code.
Step 4 — Linker: Executable File (.exe) — combines object code with library code.
Step 5 — Loader: In-Memory Image (RAM) — loads program into memory.
Step 6 — Execution: Output/Results (on screen).

📌 Section 5 — Increment/Decrement, Assignment Operators, Escape Sequences
Sec5·Q1 Concept What is the difference between prefix increment (++x) and postfix increment (x++)?
✅ ANSWER
Prefix (++x): Increment x first, then return the new value of x.
Postfix (x++): Copy the current value of x, then increment x, then return the copy (old value).
Example: if x = 5,
  int y = ++x; → y = 6, x = 6
  int y = x++; → y = 5, x = 6
Sec5·Q2 MCQ What is the output of the following code?

cout << ++x << --y << endl;
A55
B64
C46
D45
✅ ANSWER
B — 64. ++x (prefix) increments x to 6, then outputs 6. --y (prefix) decrements y to 4, then outputs 4. Output: 64.
Sec5·Q3 MCQ What is the output of the following code?

z = x++ + y--;

A10
B11

D12
✅ ANSWER
A — 10. After x=++x: x=6. After y=--y: y=4. z = x++ + y-- = 6 + 4 = 10 (then x becomes 7, y becomes 3). Output: 10.
Sec5·Q4 MCQ What is the output of the following code?

Sec5·Q5 Concept What is the const keyword in C++? What are the rules for declaring a constant variable?
✅ ANSWER
The const keyword declares read-only entities. A const variable:
• Must be initialized at the time of declaration.
• Its value cannot be modified after initialization.

Examples:
const int var; ❌ (no initialization — error)
const int var; var = 5; ❌ (can't assign after declaration)
const int var = 5; ✅ (correct)
Sec5·Q6 Concept Name the five compound assignment operators in C++ and give the syntax and equivalent expression for each.
✅ ANSWER
1. a += b  →  a = a + b (Addition assignment)
2. a -= b  →  a = a - b (Subtraction assignment)
3. a *= b  →  a = a * b (Multiplication assignment)
4. a /= b  →  a = a / b (Division assignment)
5. a %= b  →  a = a % b (Modulus assignment)

🎓 Lecture 1 — Overview of Programming Languages & Processing a C++ Program
L1·Q1 Concept List five things you should remember about computer basics from Lecture 1 (from the "Let's Remember" section).
✅ ANSWER
1. A computer is an electronic device capable of performing arithmetic and logical operations.
2. A computer system has two components: hardware and software.

L1·Q2 Concept What are the 6 steps in processing a C++ program? Name the tool used at each step.
✅ ANSWER
Step 1: Editor — creates the source program (.cpp)
Step 2: Preprocessor — processes # directives
Step 3: Compiler — checks syntax, creates object program (.obj); syntax errors reported here
Step 4: Linker — combines object program with library code → executable (.exe)
Step 5: Loader — loads executable into main memory (RAM)
Step 6: Execution — CPU runs the program
L1·Q3 Concept What is assembly language and how is it different from machine language?
✅ ANSWER
Machine language: The most basic language — a sequence of 0s and 1s that the CPU directly understands.
Assembly language: Uses easy-to-remember instructions called mnemonics (like MOV, ADD) that represent machine language instructions.
An assembler translates assembly language into machine language.
🎓 Lecture 3 — Basic Elements of C++: Data Types, Operators
L3·Q1 Concept What is a token in C++? List the three categories of tokens.
✅ ANSWER
A token is the smallest individual unit of a program. The three categories are:
1. Special symbols (e.g., +, -, *, /, ;, ==)
2. Word symbols / Reserved words (e.g., int, float, return, if)
3. Identifiers (user-defined names like variables and functions)
L3·Q2 Concept What are the rules for valid identifiers in C++? Provide examples of legal and illegal identifiers.
✅ ANSWER
Rules:
• Must consist of letters, digits, and underscore (_)
• Must begin with a letter or underscore (NOT a digit)
• C++ is case-sensitive (NUMBER ≠ number)
• Cannot be a reserved word

Legal: first, conversion, payRate, _count
Illegal:
• employee Salary — space not allowed
• Hello! — ! not allowed
• one+two — + not allowed
• 2nd — cannot begin with digit
L3·Q3 Concept List the three categories of C++ data types and explain the three sub-categories of Simple data types.
✅ ANSWER
Three categories of C++ data types:
1. Simple, 2. Structured, 3. Pointers

Simple data types have 3 sub-categories:
• Integral: integers (whole numbers without decimal) — includes char, short, int, long, bool, unsigned variants
• Floating-point: decimal numbers — float, double, long double
• Enumeration: user-defined data type

L3·Q5 Concept What is the char data type? What values can it hold? How are char literals written?
✅ ANSWER
char is the smallest integral data type; it represents a single character only.
It is used for letters, digits, and special symbols.
Char literals are enclosed in single quotation marks: 'A', 'a', '0', '*', '+', '$'
A blank space is written as: ' ' (space between single quotes)
Note: 'abc' and '!=' are NOT of type char (they are multi-character).
L3·Q6 Concept What are the three floating-point data types in C++? What is the range and precision of float vs double?
✅ ANSWER
Three floating-point types: float, double, long double

float:
• Range: −3.4E+38 to 3.4E+38
• Storage: 4 bytes
• Precision: 6–7 significant digits (single precision)

double:
• Range: −1.7E+308 to 1.7E+308
• Storage: 8 bytes
• Precision: 15 significant digits (double precision)
L3·Q7 Concept Explain the order of precedence (operator precedence) in C++ arithmetic. Give an example.
✅ ANSWER
Order of evaluation (highest to lowest priority):
1. ( ) — parentheses evaluated first
2. *, /, % — same level, evaluated left to right
3. +, − — same level, evaluated left to right

Example: 3 * 7 - 6 + 2 * 5 / 4 + 6
= ((3*7) − 6) + ((2*5)/4) + 6
= (21 − 6) + (10/4) + 6
= 15 + 2 + 6 = 23
L3·Q8 Concept What is a mixed expression in C++? What rules apply when evaluating it?
✅ ANSWER
A mixed expression has operands of different data types (e.g., integers and floating-point).

Rules:
• If an operator has operands of the same type → evaluated according to that type.
• If an operator has both int and float operands → the integer is implicitly converted to floating-point, and the result is floating-point.
• The entire expression is evaluated according to precedence rules.

Examples: 2 + 3.5 → 5.5 (float), 6/4 → 1 (int division), 6/4.0 → 1.5 (float)

✅ ANSWER

L3·Q10 Concept What is a unary operator? List three examples from C++.
✅ ANSWER
A unary operator requires only one operand and performs an action on a single value to produce a result.

Examples:
1. Logical NOT (!): Reverses a boolean value. !true → false
2. Increment/Decrement (++, --): Increases or decreases by 1. x++, --y
3. Unary Minus (−): Negates a positive number. -5
🎓 Lecture 5 — Strings in C++
L5·Q1 Concept What is a string in C++? What header file must be included to use strings?
✅ ANSWER
In C++, a string is an object that represents a sequence of characters.
You must include #include <string> to use the string type.

Advantages over char[]:
• Automatic memory management (no buffer overflows)
• Dynamic sizing (grows as you add text)
• Built-in functions for searching, replacing, and measuring
L5·Q2 Concept What is the difference between using cin and getline() for string input?
✅ ANSWER
cin >> variable: Stops at the first whitespace (space, tab, newline). Only reads a single word.

getline(cin, variable): Reads the whole line including spaces. Preferred for full names or sentences.

Example: If input is "Ahmed Ali",
cin >> name; → stores "Ahmed" only
getline(cin, name); → stores "Ahmed Ali"
L5·Q3 Concept How do you concatenate strings in C++? How do you find the length of a string? How do you access individual characters?
✅ ANSWER
Concatenation: Use the + operator.
string message = greeting + ", " + name + "!";

Length: Use .length() or .size()
cout << name.length(); — for "Alice" → outputs 5

Accessing characters: Strings act like arrays; use index.
name[0] → first character (e.g., 'A' for "Alice")

🎓 Lecture 6 — Control Structures I: if Statement & Relational/Logical Operators
L6·Q1 Concept What are the three forms of the if statement in C++? Describe each with its general syntax.
✅ ANSWER
1. One-way if:
if (condition)
    statement;
Executes statement only if condition is true.

2. Two-way if-else:
if (condition)
    statement1;
else
    statement2;
Executes statement1 if true, statement2 if false.

3. Multiple-alternative if-else if:
if (condition1)      statement1;
else if (condition2) statement2;
else if (condition3) statement3;
else                 statement4;
L6·Q2 Concept List the six relational operators in C++ with their meaning.
✅ ANSWER
== — Equal to
!= — Not equal to
<  — Less than
>  — Greater than
<= — Less than or equal to
>= — Greater than or equal to

These return a boolean value (true or false).
L6·Q3 Concept List the three logical operators in C++ and describe what each does. Provide truth table results for each.
✅ ANSWER
1. ! (NOT): Reverses the boolean value. !true=false, !false=true
2. && (AND): Returns true only if BOTH operands are true.
  true && true = true
  true && false = false
  false && true = false
  false && false = false
3. || (OR): Returns true if AT LEAST ONE operand is true.
  true || false = true
  false || false = false

L6·Q9 Concept What is the "dangling else" problem in C++?
✅ ANSWER
The dangling else problem occurs when there are multiple if statements and it is unclear which if an else belongs to. In C++, an else always matches the nearest preceding unmatched if.
if (x > 0)
    if (x > 10)
        cout << "Big";
else              // this else matches the inner if (x>10)
    cout << "Zero or negative";
Solution: Always use curly braces { } to make the code clear.

L3·EX3 Expression Evaluate: double result = 5 / 2; — What is the value? Is it 2 or 2.5?
✅ ANSWER
2.0 (NOT 2.5!)
Reason: Even though result is declared as double, both operands (5 and 2) are integers. Integer division is performed first: 5/2 = 2, then 2 is converted to 2.0 and stored in result.
To get 2.5, write: 5.0 / 2 or 5 / 2.0 or (double)5 / 2
L3·EX4 Expression Evaluate: 3 * 7 - 6 + 2 * 5 / 4 + 6 — What is the result?
✅ ANSWER
Step by step (left to right, following precedence):
= (3*7) − 6 + (2*5)/4 + 6
= 21 − 6 + 10/4 + 6
= 21 − 6 + 2 + 6   (integer division: 10/4=2)
= 23
🎓 Lecture 3 — Variable Declarations & C++ Program Structure
L3·Q11 Concept Explain the parts of a C++ variable declaration: double length;. What does this statement do?
✅ ANSWER
The statement double length; is a variable declaration. It:
1. Tells the system to allocate a memory space (8 bytes for double)
2. Names that memory space "length"
3. Specifies the data type (double — can hold decimal values)

After this, you can store values with an assignment: length = 6.0;

L3·Q13 Concept What is the purpose of using namespace std; in a C++ program?
✅ ANSWER
using namespace std; allows you to use standard library names like cout, cin, endl, and string directly without the std:: prefix.

Without it, you would need to write:
std::cout << "Hello" << std::endl;
With it, you can write:
cout << "Hello" << endl;
